
# 🏦 Streamlit Banking System

A simple **Banking System** built with **Python** and **Streamlit**.  
This project simulates basic banking operations such as:

- Creating an account  
- Depositing money  
- Withdrawing money  
- Viewing account details  
- Updating account information  
- Deleting an account  

All data is stored in a local `data.json` file.

---

## 🚀 Features
- **User-friendly Interface** using [Streamlit](https://streamlit.io/)
- **JSON-based storage** (no database required)
- Account **number generation with random ID**
- Secure 4-digit **PIN authentication**
- Real-time balance updates
- Simple **CRUD operations**
